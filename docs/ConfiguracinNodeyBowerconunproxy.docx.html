Configuración npm y Bower con un proxy

Node.js

  1. npm config set strict-ssl false
  2. npm config set registry "http://registry.npmjs.org/"

Para instalar un paquete

  1. npm --proxy http://username:password@proxy.empresa.es:8080 install nombre_del_paquete

Bower

  1. Crear en el directorio del proyecto el archivo “.bowerrc” con estos contenidos:

{

  "proxy":"http://username:password@proxy.empresa.es:8080",

  "https-proxy":"http://username:password@proxy.empresa.es:8080"

}

  1. Asegurarse de que en el directorio del proyecto existe el archivo bower.json con al menos estos contenidos:

{

    "name": "prueba",

    "version": "1.0.0",

    "dependencies": {

    },

    "devDependencies": {

    }

}
